iOS原理(2) Objective-C对象的分类

发表于 2019-10-10 | 分类于 iOS原理
1. instance 对象（实例对象）

instance对象在内存中保存：

  • isa指针
  • 其他成员变量（_age=18、_name=22，主要存储每个==成员变量的值==）
2. Class对象

class对象在内存中保存：

  • isa指针
  • superClass指针
  • 类的属性信息（@property）、类的对象方法信息
  • 类的协议信息（@protocol）、类的成员变量信息（这里的成员变量信息指的是==成员变量的描述信息==，比如：类型、名字，不包含值）
3. meta-class 对象（元类对象）

元类对象和class对象内存结构相同（它也包含class对象的其他内容，只不过为空），用途不同，内存中存储的信息主要有：

  • isa指针
  • superclass指针
  • 类的类方法信息

注意事项：

  • 元类对象的获取方式：Class objectMetaClass = object_getClass([NSObject class]);
  • 每个类在内存中有且只有一个meta-class对象

4.Object_getClass、objc_getClass、class方法的区别

Class objc_getClass(const char *aClassName): 传入字符串类名,返回对应的类对象

Class object_getClass(id obj):传入的obj可能是instance对象、class对象、meta-class对象,如果是instance对象，返回class对象。如果是class对象，返回meta-class对象。如果是meta-class对象，返回NSObject（基类）的meta-class对象。

- (Class)class、+ (Class)class:返回的就是类对象

5. sizeOf并不是一个函数，而是一个运算符
